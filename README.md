# PrompteenStudio

**Prompteen — Advanced Prompts Studio (V1.1-Beta AFL QStudios-Product)**  
A modern Python + PyQt6 application for managing, editing, and injecting prompts with a powerful syntax-aware editor.

---

## 🚀 Why PrompteenStudio Matters

PrompteenStudio is designed to **boost productivity for developers, researchers, and AI enthusiasts** by providing a structured environment for prompt engineering.  
It combines **syntax highlighting, smart injections, and clipboard-ready exports** to streamline workflows.  

This program is important because:
- It **organizes prompts** into reusable templates.
- It **injects dynamic content** seamlessly into prompt structures.
- It **supports contributors** with a clear, modular codebase.
- It **enhances GitHub SEO visibility** by being a unique open-source prompt management tool.

By contributing, you help shape a tool that empowers the growing field of **prompt engineering and AI-assisted development**.

---

## ✨ Features

- **Custom Prompt Editor** with line numbers and syntax highlighting.
- **Smart Injection System** (`{{inject}}` placeholders).
- **Clipboard Integration** for fast copy-paste workflows.
- **Explorer Pane** for managing `.prompt` files.
- **Injection Slots** for reusable content blocks.
- **Dark UI Theme** optimized for focus and readability.
- **Hotkeys** for quick actions (Save, Copy, Apply, Mark Done, etc.).

## 🛠️ Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/ExcelenceCodes/PrompteenStudio.git
cd PrompteenStudio
pip install -r requirements.txt
```bash
